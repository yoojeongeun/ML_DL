2. EDA¶

  • mnist 데이터 셋의 숫자 이미지들은 흑백 이미지
  • 각 이미지의 크기는 28 x 28
  • 각 이미지의 픽셀 값은 0 ~ 255 사이
  • 픽셀 값이 0으로 가까울 수록 어둡고, 255에 가까울 수록 밝다.
